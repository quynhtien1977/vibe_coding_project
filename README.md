# Vietnamese Teachers' Day App

A premium card creator application built with React, Tailwind CSS, and Node.js.

## Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher)

## Setup Instructions

Since the automatic setup could not be completed, please run the following commands in your terminal:

### 1. Setup Backend (Server)

Open a terminal in the `server` directory:

```bash
cd server
npm install
npm run dev
```

The server will start on `http://localhost:3000`.

### 2. Setup Frontend (Client)

Open a new terminal in the `client` directory:

```bash
cd client
npm install
npm run dev
```

The frontend will start on `http://localhost:5173`.

## Features

- **Card Gallery**: View all created cards.
- **Card Creator**: Create personalized cards with custom messages and backgrounds.
- **Premium Design**: Glassmorphism effects and smooth animations.
